## 🌟 About

A **modern Password Generator** desktop app built with **Python + Tkinter**.
Generate strong, random passwords instantly — clean dark UI, zero setup needed.

<img src="https://user-images.githubusercontent.com/73097560/115834477-dbab4500-a447-11eb-908a-139a6edaec5c.gif" width="100%" />

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🔐 Secure Generation | Strong random passwords every time |
| 📏 Custom Length | Set any length from 4 to 100+ |
| 🔡 Charset Control | Uppercase · Lowercase · Numbers · Symbols |
| 📋 Copy to Clipboard | One-click instant copy |
| ⚠️ Input Validation | Handles all invalid input gracefully |
| 🌑 Dark Theme | Premium modern dark UI |
| ⚡ Zero Dependencies | Pure Python stdlib — no pip needed |

<img src="https://user-images.githubusercontent.com/73097560/115834477-dbab4500-a447-11eb-908a-139a6edaec5c.gif" width="100%" />

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/dhruvprajapati6/Password-Generator.git

# Enter the folder
cd Password-Generator

# Run the app
python main.py
```

> 💡 Just Python 3 — no extra packages needed!
