# 🔐 Password Generator (Python GUI)

> ⚡ Generate strong, random, and secure passwords instantly with a clean and user-friendly interface.

---

## 📌 Overview

This is a **GUI-based Password Generator** built using Python.
It helps users create secure passwords with customizable length and copy them easily.

---

## 🚀 Features

* 🔢 User-defined password length
* 🔐 Strong random password generation
* 📋 Copy password to clipboard
* ⚠️ Input validation (only numbers allowed)
* 🎨 Clean and modern dark-themed UI

---

## 🖥️ Tech Stack

* Python 🐍
* Tkinter (GUI Library)
